# Тесты для лабораторной работы №02

## Входные данные
массив целых чисел

## Выходные данные
максимальное значение между (A[0]+A[n-1], A[1]+A[n-2], ...,)

## Позитивные тесты:
- 01 - все числа равны;
- 02 - только одно число;
- 03 - обычный тест;

## Негативные тесты:
- 01 - в качестве последнего элемента, вместо числа идёт буква;
- 02 - все буквы;
